# Project WRT Dashboard

This is the high-performance React dashboard for the Project WRT Object Detection system.

## Performance Stack

- **Vite**: Ultra-fast build tool and dev server.
- **TypeScript**: Type-safe development.
- **React**: Modern component-based UI.
- **Shadcn UI**: Premium component architecture.
- **Tailwind CSS**: Utility-first styling with racing design tokens.

## Development

```sh
# Install dependencies
npm install

# Start the dev server
npm run dev
```

## Dashboard Features

- **M-Tech HUD**: Live detection overlay with digital crosshairs.
- **Sector Log**: Dynamic sidebar for entity tracking.
- **Ultra-Low Latency**: Optimized for real-time visual feedback.
